# Asset Pipeline

This folder is reserved for replaceable authored art.

Current game art is generated by scripts so gameplay can keep moving. Final or intermediate sprite sheets should land here and then replace procedural nodes scene by scene.

## Suggested Layout

- `characters/pizza_fish_taco/`
  - `pizza_fish_taco_idle.png`
  - `source/pizza_fish_taco_idle_source.jpg`
  - `source/pizza_fish_taco_animation_sheet_source.jpg`
  - `frames/*.png`
  - `pizza_fish_taco.png`
  - `pizza_fish_taco.json` or Godot atlas resource
  - animation notes for idle, run, jump, fall, hurt, death, light combo, heavy, and Sauce Spin
- `enemies/burger_grunt/`
- `enemies/fry_goblin/`
- `bosses/big_bad_burger/`
- `levels/snack_city_streets/`
- `ui/`

## Player Animation Contract

`scripts/Player.gd` currently calls these animation names:

- `idle`
- `run`
- `jump`
- `fall`
- `hurt`
- `death`
- `light_1`
- `light_2`
- `light_3`
- `heavy`
- `sauce_spin`

The temporary procedural version lives in `scripts/PlayerArt.gd`. A future `AnimatedSprite2D` or `AnimationPlayer` scene should preserve the same animation names so combat code does not need to change.

## Current Imported Art

`scripts/PlayerArt.gd` now prefers sliced frame PNGs in `characters/pizza_fish_taco/frames/` and falls back to procedural shapes if those files are missing.

Current frame names:

- `idle_0`, `idle_1`, `idle_2`
- `run_0`, `run_1`, `run_2`, `run_3`
- `bite_0`, `bite_1`
- `sauce_0`
- `spin_0`, `spin_1`
- `hurt_0`, `crouch_0`, `jump_0`

To regenerate the older single idle transparent PNG:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\process_player_art.ps1 -Source 'assets\characters\pizza_fish_taco\source\pizza_fish_taco_idle_source.jpg' -Output 'assets\characters\pizza_fish_taco\pizza_fish_taco_idle.png' -TargetHeight 156
```

To regenerate the current sliced animation frames:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\slice_player_sheet.ps1 -Source 'assets\characters\pizza_fish_taco\source\pizza_fish_taco_animation_sheet_source.jpg' -OutputDir 'assets\characters\pizza_fish_taco\frames' -TargetHeight 156
```
